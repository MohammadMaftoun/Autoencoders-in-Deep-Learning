## 🧠 Autoencoders & Variational Autoencoders Collection

A hands-on deep learning project focused on understanding, implementing, and experimenting with **Autoencoders (AE)** and multiple advanced **Variational Autoencoder (VAE)** architectures using **PyTorch**.

This repository is designed for learners, researchers, and AI enthusiasts who want to explore how neural networks can compress, reconstruct, and generate data through latent space representations.

---

## 🎓 Academic Purpose

This repository was developed as part of a **Deep Learning course project** to explore the theoretical and practical aspects of Autoencoders and Variational Autoencoders.

The goal of this work is to provide both educational insights and hands-on implementations of representation learning and generative models using PyTorch.

---

## 🚀 Project Overview

This project includes:

- A complete implementation of a **basic Autoencoder**
- Multiple advanced **VAE architectures**
- Visualization and latent-space exploration
- Interactive plotting and experimentation
- Clean and educational notebook-based implementations

The notebooks are structured to make both the theory and implementation easy to follow.

---

## 📂 Included Notebooks

### 📘 `autoencoder.ipynb`
Implementation of a classic Autoencoder using PyTorch.

Features:
- Encoder / Decoder architecture
- Image reconstruction
- Loss visualization
- Training pipeline with PyTorch
- Latent representation learning

---

### 📘 `Types_of_VAE.ipynb`
A collection of advanced Variational Autoencoder models.

Implemented architectures:

- **CVAE (Conditional Variational Autoencoder)**
- **β-VAE**
- **VQ-VAE**
- **VGAE (Variational Graph Autoencoder)**

Features:
- Interactive visualizations with Plotly
- Comparative experiments
- Modular PyTorch implementations
- Educational explanations and architecture breakdowns
